# seismic-interpretation-petrel
Complete Seismic Interpretation Workflow in Petrel: well tie, Horizon & fault interpretation, attribute analysis, depth conversion

## What this repository shows

An end-to-end seismic interpretation workflow executed in Petrel, from raw
seismic and well data through to depth-converted structure maps. The work was
performed on a training dataset provided as part of the Practical Seismic
Interpretation Using Software workshop by Reservoir Solutions (RES). All
interpretation shown here is my own work.
